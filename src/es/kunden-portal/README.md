## Contribute

Contributions are welcome! 
Please read our [contribution guidelines](docs/CONTRIBUTING.md) first.  

Follow these steps:

1. Fork the repository and create a branch
2. Make changes (follow the [Coding Guidelines](docs/CODING_GUIDELINES.md))
3. Run linting
4. Submit a pull request
