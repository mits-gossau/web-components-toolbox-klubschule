# Stage Component

Stage Component at the top of a page. Available in two variants: Default & Title.

## Stage Default

<Canvas of={StageStories.StageDefault} />
<Controls of={StageStories.StageDefault} />
<Source of={StageStories.StageDefault} excludeDecorators={true} />
```html
<ks-m-stage 
    namespace="stage-default-" 
  >
    <a-picture
        picture-load 
        defaultSource="${imgUrl}" 
        alt="randomized image">
    </a-picture>
    <p class="topline">
        <ks-a-back-link href="#" alt="Alternative Text for Backlink">
            Back Link
        </ks-a-back-link>
    </p>
    <ks-a-heading tag="h1" display-2>
        ${headline}
    </ks-a-heading>
    <a-input inputid="searchField" placeholder="Label" icon-name="Search" icon-size="1.25em" search="?q=" type="search"></a-input>
</ks-m-stage>
```
