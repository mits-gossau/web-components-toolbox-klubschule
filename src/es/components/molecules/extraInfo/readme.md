## Usage

### Change Background-Color

- Add inline style to div with class `wrap`:

```html
<ks-m-extra-info>
    <div class="wrap" style="background-color: purple;"></div>
</ks-m-extra-info>
```

Hint: This component rerender html sections if API sends additional infos.
